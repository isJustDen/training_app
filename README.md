п# 💪 FitFlow

> Персональный фитнес-дневник для тренировок в зале и на улице

---

## 📱 О приложении

**FitFlow** — мобильное приложение для отслеживания тренировок, построенное на Flutter. Приложение помогает планировать тренировки, вести историю подходов, отслеживать прогресс по упражнениям и фиксировать физические замеры тела.

Разработано с нуля одним энтузиастом (Denis S. / Andromeda) для личного использования и открытого распространения.

---

## ✨ Возможности

### 🏋️ Тренировки
- Готовые шаблоны тренировок: **7 программ для зала** (фулбоди + сплит) и **4 программы калистеники** для улицы
- Создание собственных шаблонов с произвольным набором упражнений
- Генератор тренировок по группам мышц
- Поддержка **кругов / суперсетов** — объединение упражнений в группы
- Упражнения на время (планка, вис) с встроенным секундомером
- Таймер отдыха между подходами с настройкой времени

### 📊 Активная тренировка
- Пошаговое выполнение упражнений с навигацией
- Запись веса и повторений для каждого подхода
- Отображение результатов **прошлой тренировки** прямо на карточке упражнения
- Удаление ошибочно введённого подхода
- Прогресс-бар тренировки в реальном времени
- **Сохранение сессии** — незавершённая тренировка восстанавливается при повторном открытии
- Затемнение экрана в паузе между подходами (настраиваемое)

### 📈 Статистика
- Эффективность и прогресс по каждому упражнению
- История подходов с детализацией по датам
- Фильтрация по периоду (последние 30 дней)
- Удаление статистики отдельного упражнения без затрагивания шаблонов

### 📏 Замеры
- Два типа замеров: **силовые** (жим, присед, бицепс...) и **физические** (вес, объёмы тела)
- Сравнение любых двух замеров между собой
- Общий тренд прогресса за выбранный период
- Фотографии к замерам (хранятся локально)

### 🗂️ Организация
- Категории тренировок (Зал, Улица) с группами и подгруппами
- Перетаскивание упражнений внутри шаблона
- Редактирование шаблонов с защитой от случайного выхода без сохранения

### 💾 Данные
- Экспорт всех данных в JSON-файл (шаблоны, история, замеры, настройки)
- Импорт из файла резервной копии с предпросмотром содержимого
- Сброс статистики, шаблонов или категорий по отдельности
- Полный сброс до заводских настроек

### 🎨 Интерфейс
- Светлая и тёмная тема
- Анимированный экран завершения тренировки с конфетти
- Мотивационная надпись с реальным % прогресса относительно прошлой тренировки
- Звуковые оповещения и вибрация (отключаемые)
- Единый стиль шапки на всех экранах (сворачивающийся SliverAppBar)

## 📋 Начальные данные

При первом запуске приложение автоматически создаёт:

**Категория «Зал 🏋️»**
- Группа «Фулбоди»: FB Акцент грудь / спина / ноги+плечи
- Группа «Сплит»: Грудь+Трицепс / Спина+Бицепс / Ноги / Плечи

**Категория «Улица 🌳»**
- Группа «Калистеника»: Верх тела / Ноги+кор / Фулбоди
- Группа «Турник»: Интенсив

База упражнений содержит **~80 позиций** с разметкой по группам мышц: грудь, спина, плечи, бицепс, трицепс, ноги, пресс, калистеника и другие.
